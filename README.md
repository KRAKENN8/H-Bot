# H-Bot

Open-source ticket management platform.

## Features

- Telegram Integration
- Discord Integration
- Web Dashboard
- REST API
- Role Management
- Real-time Notifications
- Multi-project Support

## Tech Stack

- Node.js
- Express
- PostgreSQL
- React
- Discord.js
- Telegram Bot API
